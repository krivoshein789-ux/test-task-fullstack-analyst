#Задача 1 - Сделайте модель в нотации BPMN 2.0

## Описание
Модель бизнес-процесса выдачи IT-оборудования пользователю по заявке.

Диаграмма выполнена в нотации BPMN 2.0 на основе предоставленного описания процесса.

## Что отображено в модели
- создание заявки пользователем;
- регистрация заявок в системах №1 и №2;
- проверка наличия оборудования;
- сценарий закупки оборудования;
- взаимодействие с поставщиком;
- логистика доставки оборудования;
- выдача оборудования пользователю;
- закрытие заявок;
- оценка качества сервиса.

## Участники
- Пользователь
- Техническая поддержка
- Снабжение
- Логистика
- Поставщик

## Дополнительные замечания
В процессе были выявлены неопределенности:
- не до конца понятно создает ли пользователь сам заявку в системе №1 или заявка создается сотрудником технической поддержки
- не описан сценарий отмены заявки;
- не описан сценарий отказа поставщика.
