## User Story

Как продавец,
я хочу публиковать свои товары через личный кабинет,
чтобы покупатели могли видеть их на витрине маркетплейса и приобретать.

## Acceptance criteria

- Продавец может создать новый товар через личный кабинет
- Продавец может указать:
    - Название товара
    - Описание
    - Цену
    - Категорию
    - Количество товаров
- Продавец может загрузить изображение товара
- Система проверяет корректность данных
- После успешной публикации товар отображается на витрине маркетплейса
- В случае ошибки система отображает сообщение пользователю
