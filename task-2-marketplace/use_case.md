# Use Case — Публикация товара

## Название
Публикация товара на маркетплейсе

## Цель
Позволить продавцу разместить товар на витрине маркетплейса.

## Актор
Продавец

## Предусловия
- Продавец зарегистрирован в системе
- Продавец авторизован в личном кабинете

## Триггер
Продавец нажимает кнопку "Добавить товар"

## Основной сценарий
1. Продавец открывает раздел "Мои товары"
2. Продавец нажимает кнопку "Добавить товар"
3. Система отображает форму создания товара
4. Продавец вводит данные товара:
   - название
   - описание
   - цену
   - категорию
   - количество товара
5. Продавец загружает изображения товара
6. Продавец нажимает кнопку "Опубликовать"
7. Система выполняет проверку данных
8. Система сохраняет товар
9. Система публикует товар на витрине маркетплейса
10. Система отображает сообщение об успешной публикации

## Альтернативные сценарии

### 1. Не заполнены обязательные поля
1) Система отображает сообщение об ошибке
2) Система подсвечивает незаполненные поля
3) Публикация товара не выполняется

### 2. Указана некорректная цена
1) Система отображает сообщение об ошибке
2) Пользователь должен указать корректную цену

### 3. Загружен неподдерживаемый формат изображения
1) Система отображает сообщение об ошибке
2) Пользователь должен загрузить изображение допустимого формата

### 4. Ошибка сохранения товара
1) Система отображает сообщение о временной ошибке
2) Публикация товара не выполняется

## Постусловия
- Товар сохранён в системе
- Товар отображается на витрине маркетплейса
- Товар доступен покупателям к приобретению
